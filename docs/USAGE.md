# 🚀 BLUX Usage Guide

## 🔥 Terminal Commands:
- /recall notes → View notes
- /note [text] → Save note
- exit → Quit

## 🗣️ Voice Commands:
- Say 'BLUX' to activate
- Speak any natural language task

## 🌐 Web UI:
Visit http://localhost:5000 on your phone browser.

## 🔗 Automations:
- NFC Trigger → Runs start_blux.sh
- Bluetooth Proximity → Starts BLUX session
- Git Sync Automation → Push local commits

## 🧠 Memory:
Stored in memory/memory.json
→ Supports notes, tasks, history
